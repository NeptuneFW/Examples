# Example Blog


### A blog script with Neptune Framework Beta V1

### Requirements

  - PHP >= 5.6
  - Composer

### Installation

Clone or download this repository. 
And go to your project directory

```sh
$ composer install
$ composer dump-autoload -o
$ php neptune --secret
```

And execute blog.sql in your mysql server
